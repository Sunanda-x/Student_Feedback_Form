# Student Feedback Form

## 🖼️ Screenshots

### 🏠 Homepage
This is the first page of the system. It shows the institute name and gives two options: students can fill feedback, and principals can log in.
![Homepage Screenshot](Screenshots/Homepage.png)

### 📝 Fill Feedback
This is the form where students write and submit their feedback. It’s simple and easy to use.
![Fill Feedback Screenshot](Screenshots/Fill_Feedback.png)

### 🔑 Principal Login
This page is for the principal to log in securely. It has username and password fields, plus a “Forgot Password” option.
![Principal Login Screenshot](Screenshots/Principal_login.png)

### 🗂️ Principal Dashboard
This page shows all the feedback in a table. The principal can read and manage student responses here.
![Principal Dashboard Screenshot](Screenshots/Principal_dashboard.png)

### 📊 Analytics
This page shows a summary of all feedback. The principal can quickly see results and understand what students think.
![Analytics Screenshot](Screenshots/Analytics.png)
